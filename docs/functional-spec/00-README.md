# Especificacion Funcional - Producto Unico

Estado: Borrador v1 (conceptual)
Fecha: 2026-03-14

## Proposito

Este paquete define la vision funcional del producto para un solo negocio, sin implementacion tecnica detallada. El objetivo es alinear alcance, reglas y prioridades antes de desarrollo.

## Alcance del paquete

- Vision de producto y objetivos de negocio.
- Alcance MVP y limites fuera de alcance.
- Actores, casos de uso y permisos.
- Modelo de datos conceptual.
- Contrato API conceptual y semantica de errores.
- Criterios de exito y roadmap.

## Como leer estos documentos

1. Leer `01-vision-producto.md` y `02-alcance-mvp.md`.
2. Validar `03-actores-casos-uso.md` y `04-permisos-acceso.md`.
3. Revisar `05-modelo-datos-conceptual.md` y `06-contrato-api-conceptual.md`.
4. Cerrar con `07-semantica-errores.md`, `08-roadmap-fases.md`, `09-criterios-exito.md`.
5. Usar `10-acta-fase-0.md` como evidencia de cierre conceptual.
