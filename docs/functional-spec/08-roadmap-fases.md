# Roadmap por Fases

## Fase 0 - Cierre conceptual (completada)

- Vision, alcance, actores, permisos, datos y contrato definidos a nivel funcional.

## Fase 1 - Endurecimiento MVP

- Formalizar endpoints internos.
- Implementar frontera de autenticacion/autorizacion por rol.
- Consolidar estados y trazabilidad de solicitudes.

## Fase 2 - Operacion interna

- Mejorar panel interno con filtros y notas.
- Incorporar reglas de asignacion de solicitudes.
- Estabilizar metricas operativas minimas.

## Fase 3 - Optimizacion de conversion

- Mejoras de UX en formulario y CTA.
- Ajustes de contenido para conversion.
- Medicion continua de lead-to-contact.

## Fase 4 - Escalamiento controlado

- Evolucion de reporting.
- Automatizaciones no criticas.
- Ajustes segun uso real.
