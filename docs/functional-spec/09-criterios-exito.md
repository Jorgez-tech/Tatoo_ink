# Criterios de Exito

## Exito funcional MVP

- Toda solicitud publica valida queda registrada.
- Equipo interno puede listar, revisar y actualizar estado de solicitudes.
- Permisos por rol aplican correctamente en zona interna.

## Exito operativo

- Trazabilidad minima de cambios de estado y notas.
- Respuestas de error consistentes para cliente y soporte.
- Tiempo de respuesta inicial al cliente potencial mejora respecto al estado previo.

## Indicadores iniciales (conceptuales)

- Tasa de conversion visitante -> solicitud.
- Tiempo promedio primera respuesta.
- Porcentaje de solicitudes sin seguimiento.
- Distribucion de estados por periodo.

## Criterio de salida de fase conceptual

Se considera cerrada cuando negocio y equipo tecnico validan estos documentos como base suficiente para planificar implementacion.
