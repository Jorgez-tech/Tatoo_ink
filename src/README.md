# Frontend - Ink Studio

Este directorio contiene el frontend (React + TypeScript + Vite).

## Requisitos

- Node.js 18+

## Desarrollo

Desde la raíz del repositorio:

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Lint

```bash
npm run lint
```

## Configuración

El frontend consume el backend usando:

- `VITE_API_BASE_URL`

Ver:

- `docs/BACKEND-INTEGRATION.md`
- `docs/API-REST.md`
